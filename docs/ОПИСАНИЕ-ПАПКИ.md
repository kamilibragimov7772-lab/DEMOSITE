# TERMY-site-polish — что где лежит

**Назначение папки:** рабочее пространство для проекта «сделать из сырого сайта TERMY (репо `bot-ponton`) конфетку». Сюда стекаются исходники от клиента, сам склонированный репо сайта, черновики правок, финальные результаты и референсы.

**Актуально на:** 2026-04-11
**Общий размер:** ~70 МБ
**Клиент:** ООО «ТЕРМИ», бренд Termy (термоупаковка)

## 📁 02_github_repo — клон репо сайта `bot-ponton`

**Размер:** 37 МБ
**GitHub:** https://github.com/kamilibragimov7772-lab/bot-ponton
**Деплой:** GitHub Pages через Actions `static.yml`, ветка `main`

Это **рабочая копия сайта Termy**, куда вносятся все правки. Это источник правды — что здесь лежит, то попадает на продакшен после `git push`.
