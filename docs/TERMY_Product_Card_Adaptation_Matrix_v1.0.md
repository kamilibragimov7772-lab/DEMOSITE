## Ключевые выводы для немедленного применения к сайту

### A. Что нужно добавить в плитки каталога на главной (`/index.html`, `.pcard`):

1. **Performance badge (блок 1.5) — универсально для всех категорий**, не только SHOCK:
   - CAT-01/02 Термопакеты: «Трёхслойная изоляция · до N ч» (пока нет точных данных — fallback «надёжная термозащита»)
   - CAT-03 SHOCK: уже есть (темп. диапазон)
   - CAT-04 Бабл: «Удар + температура»
   - CAT-05 Лайнеры: «Совместимо с коробами N×M»
   - CAT-06 Сумки: «Многоразовая · брендирование»
   - CAT-07 Термосы: «Брендирование · корпоративный мерч»

2. **Унифицированная структура плитки** (уже близка к нужной):
   ```
   img → cat-tag → name → [temp-badge?] → desc (1 строка benefit-first) → 2 spec-метки → price → CTA
   ```
   Сейчас desc часто feature-based («Мет/ВПЭ/Мет») — переписать на benefit-first («Надёжная термозащита до 7 ч»).

3. **Блокер:** ссылки `/products/*/` ведут в никуда (папка не существует). Нужны либо детальные страницы, либо временный редирект на `#product-grid` с якорями.

### B. Configuration target по категориям:

| Категория | Плитка (сейчас) | Детальная страница (надо) |
|---|---|---|
| CAT-01, CAT-02 Термопакеты | Standard | Extended |
| CAT-03 SHOCK | Standard (есть temp) | Extended |
| CAT-04 Бабл | Minimum | Standard |
| CAT-05 Лайнеры | Standard | Extended |
| CAT-06 Сумки | Standard | Standard |
| CAT-07 Термосы | Minimum | Minimum |

### C. Publishing blockers по нашему сайту на 2026-04-14:

- Детальные страницы продуктов **не существуют** (папка `/products/` отсутствует)
- Время удержания температуры — нет данных для большинства SKU
- Сертификаты — не оцифрованы в виде файлов к скачиванию
- Фото в контексте (с товаром внутри) — есть только студийные
