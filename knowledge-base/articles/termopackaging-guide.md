# Гайд: Как выбрать термоупаковку TERMY

## Для кого этот гайд
Для менеджеров по закупкам и логистов, которые подбирают термоупаковку под конкретную задачу.

## Шаг 1: Определите тип груза
- **Охлаждённые продукты (+2…+8°C)** → SHOCK® Голубой + термопакет Standart/PRO
- **Замороженные продукты (0…−15°C)** → SHOCK® Зелёный + термопакет PRO или LinerBox®
- **Деликатные грузы (~0°C)** → SHOCK® Фиолетовый (гель) + термопакет Standart
- **Без строгого температурного режима** → термопакет Light или БаблПак

## Шаг 2: Определите время в пути
- **До 2 часов** → термопакет Light/Standart, SHOCK® 300 мл
- **2–6 часов** → термопакет Standart/PRO, SHOCK® 300–450 мл
- **6–12 часов** → термопакет PRO, SHOCK® 450–600 мл
- **12–24 часа** → LinerBox® + SHOCK® 600–1000 мл
- **24+ часов** → LinerBox® + SHOCK® 1000 мл × 3–4 шт

## Шаг 3: Учтите сезон
- **Зима (−5…−15°C)** → можно уменьшить количество АКХ на 20–30%
- **Лето (+18…+28°C)** → увеличьте на 10–20%
- **Жаркое лето (+30…+40°C)** → увеличьте на 30–50%

## Шаг 4: Выберите формат упаковки
- **Термопакет** → для заказов до 15 кг, маршруты до 7 часов
- **Термошоппер** → для выдачи клиентам, многоразовый, до 2 часов
- **LinerBox®** → для коробочных отгрузок, до 72 часов, до 25 кг
- **БаблПак** → дополнительная защита деликатных вложений

## Ключевые серии термопакетов
| Серия | Время | Назначение |
|-------|-------|-----------|
| Light | до 4 ч | Бюджетная повседневная доставка |
| Standart | до 4–7 ч | Универсальный формат |
| PRO | до 7 ч | Максимальная термоизоляция |
| С дном | до 4–7 ч | Устойчивая вертикальная сборка |
| Zip-lock | до 4–7 ч | Компактные вложения |
| Клапан | до 4–7 ч | Быстрая фиксация без завязывания |
