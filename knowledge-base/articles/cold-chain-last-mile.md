# Холодовая цепь last-mile: стандарты и практика

## Что такое last-mile в термологистике
Последняя миля — это участок доставки от склада/dark store до конечного получателя. Именно на этом участке чаще всего нарушается температурный режим, потому что груз покидает контролируемую среду.

## Типовые сценарии
1. **E-grocery**: сборка корзины с разными температурными зонами → термопакет Standart/PRO + SHOCK® 300–450 мл
2. **Курьерская доставка**: мульти-ресторанный формат → термошопперы + термопакеты
3. **Фармацевтика**: термолабильные препараты → LinerBox® + SHOCK® Голубой
4. **Кейтеринг**: выездное обслуживание → термопакеты PRO + LinerBox® + SHOCK®

## Как подобрать решение
- Определите температурный режим груза
- Оцените время доставки (включая ожидание)
- Учтите сезон и температуру окружающей среды
- Выберите формат упаковки (пакет / шоппер / короб)
- Подберите аккумуляторы SHOCK® по объёму и режиму

## Сезонная корректировка
Летом (>+25°C) рекомендуется увеличивать количество аккумуляторов на 30–50% или переходить на больший объём. Зимой (<0°C) можно сократить на 20–30%.

## Продукция TERMY для last-mile
- **Термопакеты** (6 серий): основной формат для коротких и средних маршрутов
- **Термошопперы**: многоразовый формат выдачи, 10+ циклов
- **SHOCK®**: аккумуляторы с 3 режимами для точной стабилизации температуры
- **LinerBox®**: для длинных маршрутов и коробочных отгрузок до 72 часов
