## Общий вывод

- **Все фото на белом фоне** — нужен batch removebg
- **Все выглядят как 3D-рендеры** (profi packshot) — высокое качество, края чёткие → removebg отработает хорошо
- **Одна аномалия** — `termopaket-42x45.webp` сдвоенный кадр. Надо сначала кроп, потом removebg
- **Тени под продуктом** есть у SHOCK/LinerBox/шоппера — для фиолетовой hero-композиции лучше убрать и добавить свои CSS `drop-shadow()` (уже есть в стилях)
